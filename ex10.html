Function

Basic

코드:function list () { }
리스트가 일정하지 않게 반복되면 반복문을 사용할 수 없다. -> 함수(Function) 사용.



Parameter & Argumnet

코드: function 함수이름(parameter) {document.write();}
함수이름(argument);


함수 안에 document.write 코드를 사용하는 것은, 해당 코드에 알맞은 값을 단지 웹에 출력해 주는 것 일뿐. 이 함수를 다른 용도로 사용하지 못한다.
따라서 style속성을 지정하려면 원하는 갯수만큼의 함수를 만들어야 한다.



Return

코드: function sum2(left,right) {return left+right;}
document.write(sum2(7,8))


Return은 매개변수(parameter)를 통해 들어온 인자(argument)를 함수 내의 코드에 따라서 연산하여 출력함으로서, 해당 함수를 이용하여 여러가지 style속성 지정 등의 다양한 활용을 할 수 있다.
